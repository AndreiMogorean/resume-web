Bootstrap Features Used

Below is a list of all the Bootstrap features used in the main resume file:

  • Responsive Grid System: Used `container`, `row`, and `col-md-*` to create a responsive layout.
  • Typography Utilities: Classes like `text-center`, `fw-bold`, `text-muted`, and `text-primary` are applied to style text elements.
  • Buttons: Used `btn`, `btn-primary`, and modal-related classes (`data-bs-toggle`, `data-bs-target`).
  • Modal: Implemented a modal dialog (`modal`, `modal-dialog`, `modal-content`) for contact information.
  • Images: Utilized `img-fluid` to make the sidebar profile picture responsive.
  • Spacing Utilities: Classes like `mt-*`, `mb-*`, `pt-*`, `pb-*` are used to add consistent spacing.
  • Background and Text Colors: Classes like `bg-dark`, `text-white`, `bg-primary`, and `text-warning` are applied.
  • Cards and Shadow: Applied `shadow-lg` and `rounded` for card-like styling.
  • Flexbox: Used flexbox utilities for alignment, such as `d-flex` for sidebar and layout responsiveness.
  • Responsive Design: Media queries and Bootstrap's built-in responsive grid help the page adapt to different screen sizes.
